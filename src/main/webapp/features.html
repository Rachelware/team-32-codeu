Message Input

Beyond the basic writing styles, you can use standard BBCode to style your text.

  • - Bold text with [b]your text[/b]

  • - Italicize text with [i]your text[/i]

  • - Underline text with [u]your text[/u]

  • - Strikethrough text with [s]your text[/s]

  • - Quote text with [quote]your text[/quote]

  • - Monospace text with [code]your text[/code]

  • - Enter a line break with [/br]

  • - Display an image (.png or .jpg) or gif by entering a link
